🚀 배포하는 방법 (2단계):

   1️⃣  GitHub에 코드 올리기 (5분)

       cd C:\storyboard-maker
       git branch -M main
       git remote add origin https://github.com/YOUR_USERNAME/storyboard-maker.git
       git push -u origin main

   2️⃣  Vercel에 배포하기 (3분)

       웹: https://vercel.com/new
       → GitHub 로그인
       → storyboard-maker 선택
       → Deploy 클릭

   app/
   ├── page.tsx           - 메인 페이지

   ├── StoryboardContext.tsx  - 상태 관리

   └── storyboard.ts      - TypeScript 타입 정의

✨ 주요 기능:

   🎬 씬 관리
      • 여러 씬 생성, 수정, 삭제
      • 자동 씬 번호 매기기

   🎥 컷 관리
      • 각 씬에서 여러 컷 추가
      • 컷 편집 및 삭제
      • 자동 컷 번호 매기기

   📹 카메라 설정
      • 구도: 하이앵글, 아이레벨, 로우앵글, 더치앵글
      • 사이즈: LS, FS, KS, MS, BS, HS, HCU, OCU (8가지)
      • 렌즈: 24mm, 35mm, 50mm, 85mm, 105mm, 200mm
      • 무빙: 스태틱, 팬, 틸트, 돌리 인/아웃, 크레인, 줌, 트래킹, 핸드헬드

   💾 데이터 저장
      • 브라우저 localStorage에 자동 저장
      • JSON 내보내기 기능

❓ 자주 묻는 질문:

   Q: 데이터는 어디에 저장되나요?
   A: 각 사용자의 브라우저 localStorage에 저장됩니다.
      (다른 브라우저나 기기에서는 다른 데이터)

   Q: 팀 전체가 같은 데이터를 보려면?
   A: JSON 파일로 내보내서 공유하세요.
      향후 데이터베이스 추가로 실시간 협업 가능.

   Q: 배포 후 비용이 드나요?
   A: Vercel 무료 플랜은 충분합니다.
      (월간 방문자, 대역폭, 스토리지 무료)
